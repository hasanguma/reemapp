# 🌸✨ ريم Ultra (Reem AI Ultra) — مساعدة ذكية عربية متعددة الوسائط

مساعدة ذكية تعتمد على Gemini API، تفهم النصوص والصور، ترسم ما تتخيلينه، وتتحدث معكِ في مكالمة صوتية مباشرة — كل ذلك بتجربة تطبيق هاتف (PWA) أنيقة.

## ✨ الجديد في نسخة Ultra
1. **فهم الصور (Visual Understanding):** اضغطي 📷 بجانب حقل الكتابة لإرفاق صورة — تستطيع "ريم" وصفها، قراءة أي نص بداخلها، وحل التمارين والمسائل المصوَّرة خطوة بخطوة.
2. **توليد الصور (Image Generation):** اطلبي من "ريم" أن ترسم أي شيء (مثلاً: "ارسمي لي منظر غروب على الشاطئ") فتُنشئ الصورة فعلياً عبر خدمة [Pollinations.ai](https://pollinations.ai) المجانية وتعرضها مباشرة في المحادثة.
3. **أيقونة وتجربة تطبيق أصلية (PWA):** أيقونة جديدة بتدرجات نيون (بنفسجي/أزرق/ذهبي)، مع شاشة افتتاحية (Splash Screen) تظهر عند فتح التطبيق من الشاشة الرئيسية.
4. **مكالمة صوتية مباشرة (📞):** وضع محادثة صوتية متواصلة — تتحدثين وتستمع "ريم"، ثم تردّ صوتياً تلقائياً، دون الحاجة للضغط على أي زر بين الجمل.
5. **ردود نصية افتراضياً:** لم تعد الردود تُقرأ صوتياً تلقائياً في الشات العادي (لتقليل الإزعاج)؛ يمكن تفعيل القراءة التلقائية من زر 🔇/🔊 في الأعلى، أو استخدام وضع المكالمة الصوتية للحديث الكامل.
6. **تنسيق Markdown:** عناوين، قوائم، **نص عريض**، وأكواد تُعرض بتنسيق واضح داخل فقاعات الرد.
7. **أزرار نسخ 📋 وإعادة توليد 🔄** أسفل كل رد من "ريم"، بالإضافة لزر مسح المحادثة 🗑️ الموجود مسبقاً.
8. **معرفة دقيقة بالوقت والتاريخ ⏱️:** يتم حقن الوقت والتاريخ الحاليين (ميلادي + هجري تقريبي) ديناميكياً في تعليمات النظام مع كل طلب، فتجيب "ريم" فوراً وبثقة عند سؤالها "كم الساعة؟" أو "ما تاريخ اليوم؟" دون الحاجة لإعادة تشغيل السيرفر ودون أن تدّعي عدم معرفتها بالوقت.
9. **نموذج `gemini-1.5-flash-latest`:** يشير دائماً لآخر نسخة مستقرة من جوجل تلقائياً دون الحاجة لتحديث الكود يدوياً كلما صدر تحديث.
10. **إعدادات أمان مضبوطة بعناية (`safetySettings`):** فئة "المضايقة" (Harassment) مخفَّفة قليلاً (`BLOCK_ONLY_HIGH`) لتفادي حجب ردود بريئة أو أسلوب كلام عادي بالخطأ، بينما تبقى فئات "خطاب الكراهية"، "المحتوى الجنسي"، و"المحتوى الخطير" على الحماية الافتراضية الآمنة (`BLOCK_MEDIUM_AND_ABOVE`) دون أي تعطيل — حفاظاً على أمان المستخدمين وحساب الـ API.

## ⚙️ خطوات التشغيل المحلي (Localhost)

### 1) تثبيت المتطلبات
تأكد من تثبيت [Node.js](https://nodejs.org) (إصدار 18 أو أحدث) على جهازك.

### 2) تثبيت الحزم
افتح الطرفية (Terminal) داخل مجلد المشروع ونفّذ:

```bash
npm install
```

### 3) الحصول على مفتاح Gemini API
- ادخل إلى: https://aistudio.google.com/app/apikey
- أنشئ مفتاحاً مجانياً (API Key).

### 4) إعداد ملف البيئة
انسخ الملف `.env.example` وأعد تسميته إلى `.env`:

```bash
cp .env.example .env
```

ثم افتح `.env` وضع مفتاحك:

```
GEMINI_API_KEY=مفتاحك_هنا
PORT=3000
```

> ⚠️ **تنبيه أمني مهم:** لاحظنا أن ملف `.env` المرفوع معك يحتوي بالفعل على مفتاح API فعلي. بما أن هذا الملف قد يكون تمت مشاركته خارج جهازك (كملف مضغوط)، **يُنصح بشدة بإلغاء هذا المفتاح وإنشاء مفتاح جديد** من رابط AI Studio أعلاه، ثم وضع المفتاح الجديد فقط في `.env` على جهازك، ولا تشاركي هذا الملف مع أي أحد أو ترفعيه إلى GitHub.

### 5) تشغيل السيرفر

```bash
npm start
```

أو أثناء التطوير (مع إعادة التشغيل التلقائي):

```bash
npm run dev
```

### 6) فتح التطبيق
افتح المتصفح على الرابط:

```
http://localhost:3000
```

## ⏱️ معرفة الوقت والتاريخ الحاليين
كل طلب يصل إلى `/api/chat` يُبنى له نموذج جديد بتعليمات نظام تحتوي التاريخ الميلادي، التاريخ الهجري التقريبي، واليوم والوقت الحاليين (محسوبة لحظياً عبر `Intl.DateTimeFormat`، بدون أي طلب شبكة إضافي). لذلك تستطيع "ريم" الإجابة فوراً وبدقة على أسئلة مثل "كم الساعة الآن؟" أو "ما تاريخ اليوم؟" دون أي تأخير أو حاجة لإعادة تشغيل السيرفر.
- المنطقة الزمنية الافتراضية هي `Asia/Riyadh`، ويمكن تغييرها بسهولة عبر متغير `TIMEZONE` في ملف `.env` (مثال: `TIMEZONE=Africa/Cairo`).
- إن لم يدعم تقويم النظام الهجري (نادر الحدوث)، يتم تجاهل سطر التاريخ الهجري بأمان دون أي تأثير على باقي المعلومات.

## 🖼️ كيف يعمل فهم الصور وتوليدها
- **فهم صورة:** اضغطي 📷، اختاري صورة (JPG/PNG/WEBP، أقل من 8 ميجابايت)، أضيفي سؤالاً اختيارياً (مثل "حل هذه المسألة")، ثم اضغطي إرسال. تُرسل الصورة إلى `/api/chat` كـ Base64 وتُحلَّل عبر نموذج Gemini متعدد الوسائط.
- **توليد صورة:** اكتبي طلباً يتضمن كلمات مثل "ارسمي / صمّمي لي صورة / تخيّلي صورة" (أو بالإنجليزية "draw / generate an image")، فيقوم السيرفر بتحسين الوصف وإرساله إلى Pollinations.ai لإنشاء الصورة وعرضها مباشرة في المحادثة. **هذه الخدمة خارجية ومجانية ولا تتطلب مفتاح API.**

## 📞 المكالمة الصوتية المباشرة
اضغطي على 📞 في الأعلى لفتح وضع المكالمة: تتحدثين، وتستمع "ريم" تلقائياً عبر المايكروفون، ثم تردّ بصوتها. يمكن كتم المايكروفون أو إنهاء المكالمة من الأزرار الظاهرة. هذا الوضع يعتمد بالكامل على مكتبات المتصفح المجانية (SpeechRecognition + SpeechSynthesis)، لذا:
- يعمل بأفضل جودة على متصفح **Google Chrome**.
- جودة ووضوح الصوت العربي يعتمدان على الأصوات المثبتة في نظام تشغيلك؛ كلما كان النظام يحتوي أصواتاً عربية عالية الجودة (مثل أصوات Google/Microsoft الطبيعية)، كان الصوت أوضح وأكثر طبيعية.
- إن لم يدعم متصفحك خاصية التعرف على الصوت، سيظهر تنبيه بذلك عند محاولة بدء المكالمة.

## 🎙️ ملاحظات حول الصوت في الشات العادي
- الردود لم تعد تُقرأ صوتياً تلقائياً بشكل افتراضي. فعّلي ذلك من زر 🔇 (يتحول إلى 🔊 عند التفعيل) أعلى الشاشة إذا رغبتِ بسماع كل رد.
- زر المايكروفون 🎤 بجانب حقل الكتابة يحوّل كلامكِ إلى نص لإرساله كرسالة عادية (إدخال سريع بدون فتح وضع المكالمة الكاملة).

## 🔒 ملاحظات أمنية
- مفتاح `GEMINI_API_KEY` لا يظهر أبداً في الواجهة الأمامية؛ كل الاتصال بـ Gemini يتم من السيرفر فقط.
- شخصية "ريم" (System Instruction) مخزّنة في السيرفر حصراً، فلا يمكن للمستخدم تغييرها من المتصفح.
- الصور التي يرفقها المستخدم تُرسل إلى السيرفر ثم إلى Gemini للتحليل، ولا تُخزَّن دائماً على القرص (تبقى في الذاكرة أثناء الطلب فقط).
- ردود "ريم" النصية تُعرض عبر `marked.js` مع تعقيم كامل بـ `DOMPurify` لمنع أي هجمات XSS من محتوى Markdown.
- ملف `.env` مُستثنى من Git عبر `.gitignore`.

## 🛠️ حل المشاكل الشائعة
| المشكلة | الحل |
|---|---|
| السيرفر يتوقف عند التشغيل برسالة GEMINI_API_KEY | تأكد من إنشاء ملف `.env` ووضع المفتاح بداخله |
| رسالة "تعذّر التحقق من مفتاح الخدمة" (401) | المفتاح غير صحيح أو منتهي الصلاحية — أنشئ مفتاحاً جديداً من AI Studio |
| رسالة "تم الوصول للحد الأقصى من الطلبات" (429) | انتظري قليلاً؛ تم تجاوز الحصة المجانية المسموحة مؤقتاً |
| رسالة "تعذّر الوصول إلى خدمة الذكاء الاصطناعي" (503) | تحقق من اتصال السيرفر (وليس المتصفح فقط) بالإنترنت |
| الصورة المولَّدة لا تظهر | تأكد من اتصال السيرفر بالإنترنت؛ خدمة Pollinations خارجية وتحتاج اتصالاً فعلياً وقد تتأخر قليلاً في أوقات الازدحام |
| لا يوجد صوت عند الرد في الشات العادي | هذا هو السلوك الافتراضي الجديد؛ فعّلي زر 🔊 أعلى الشاشة لتفعيل القراءة التلقائية |
| زر المايكروفون أو المكالمة الصوتية غير ظاهر/لا يعمل | متصفحك لا يدعم SpeechRecognition؛ جرّبي Google Chrome |
| الأيقونة القديمة لا تزال تظهر بعد التحديث | امسحي بيانات الموقع/الكاش من إعدادات المتصفح، أو أزيلي التطبيق من الشاشة الرئيسية وأضيفيه من جديد (Service Worker محدَّث تلقائياً لحذف الكاش القديم) |
